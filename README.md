# NUrikaBE Solver (NUBE)
Authors: Simon Kim, Ji Hun Wang

Solving Nurikabe using DecisionTransformers. 

Using different techniques in RL to solve [Nurikabe](https://en.wikipedia.org/wiki/Nurikabe_(puzzle)).
